# 常用 Claude Skills 推荐

## 个人推荐

以下是我目前安装并高频使用的 Claude Skills:

### 1. Anthropic 官方 Skills
- GitHub: https://github.com/anthropics/skills
- 官方维护，质量保证，包含文档处理、代码审查、设计等核心功能

### 2. Superpowers
- GitHub: https://github.com/obra/superpowers
- ⭐ 1.6万 Star 的 Skill 精选
- 覆盖脑暴、写需求文档、开发、测试全流程，口碑极佳

### 3. Planning-with-files
- GitHub: https://github.com/OthmanAdi/planning-with-files
- 参考 Manus 的 Agent 方法编写的 Skill
- 适合多步骤任务，可指导其他 Skill 协同工作

### 4. X-article-publisher-skill
- GitHub: https://github.com/wshuyi/x-article-publisher-skill
- 王树义老师编写的 X 文章发布 Skill
- 值得研究，可扩展到其他平台的自动化发布

### 5. NotebookLM skill
- GitHub: https://github.com/PleasePrompto/notebooklm-skill
- 自动上传 PDF、YouTube 链接到 NotebookLM
- 适合 NotebookLM 内容的自动化处理

## 原创 Skills 项目

我开源了自己的 Skills 项目 **happy-claude-skills**，只放真正"好用"的精品:

### 安装方法
```bash
# 添加仓库
/plugin marketplace add iamzhihuix/happy-claude-skills

# 安装具体 Skills
/plugin install docx-format-replicator@happy-claude-skills
/plugin install video-processor@happy-claude-skills
/plugin install wechat-article-writer@happy-claude-skills
```

### 包含的 Skills

#### ✅ docx-format-replicator
完美复制 Word 文档样式。上传模板，自动提取格式，生成一样的新文档。适合报告、简历、合同批量处理。

#### ✅ video-processor
直接输入 YouTube 链接，就能转录、总结视频内容。无需其他工具或付费服务，在 Claude Code 里一键搞定视频笔记或摘要。

#### ✅ wechat-article-writer
公众号写作助手。内置快速写作流程：研究话题、生成大纲、正文、标题优化、排版建议。帮助更快写出阅读量更高的文章。
