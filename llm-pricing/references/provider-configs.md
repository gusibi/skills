# LLM 提供商页面配置参考

本文档记录各提供商网页的特征和提取策略。

## volcengine（火山引擎）

**URL**: `https://www.volcengine.com/docs/82379/1544106`

**页面特征**：
- 文档页面，包含价格表格
- 表格形式展示，可直接提取
- 价格单位：元/千 tokens

**提取策略**：
1. 定位价格表格
2. 提取表格行数据
3. 注意区分输入/输出价格列

---

## qiniu（七牛云）

**URL**: `https://www.qiniu.com/ai/models`

**页面特征**：
- 模型卡片列表
- 可能有"精选"和"所有模型"标签页
- 价格单位：元/千 tokens

**提取策略**：
1. 检查是否有标签页切换，点击"所有模型"
2. 等待页面加载完成
3. 遍历所有模型卡片提取信息

---

## siliconflow（硅基流动）

**URL**: `https://www.siliconflow.cn/pricing`

**页面特征**：
- 定价页面，可能有分类标签
- 可能需要滚动加载更多
- 价格单位：元/百万 tokens 或 元/千 tokens

**提取策略**：
1. 滚动到页面底部确保加载完整
2. 检查是否有展开按钮
3. 提取所有模型价格信息

---

## ppio

**URL**: `https://ppio.com/ai-computing/llm-api`

**页面特征**：
- 包含多种模型卡片
- 模型按类别（如“大语言”）折叠
- 价格单位：元/百万 tokens (¥/Mt)

**提取策略**：
1. 访问 https://ppio.com/ai-computing/llm-api。
2. 点击“所有模型” (All Models)。
3. 展开“大语言”类别（点击“View All”或“展开”）。
4. 等待模型列表完成加载。
5. 提取所有模型卡片信息（提供商、模型名称、输入/输出/缓存价格、上下文窗口、标签）。
6. 将结果格式化为对象列表。
---

## 通用提取字段

| 字段 | 说明 | 示例 |
|------|------|------|
| model | 模型名称 | deepseek-v3 |
| input_price | 输入价格 | 2.0 |
| output_price | 输出价格 | 8.0 |
| raw_unit | 原始单位 | per 1K tokens |
| context_length | 上下文长度 | 128000 |
| tags | 特殊标签 | ["满血版", "推理模型"] |

## 常见标签识别

- **满血版** / **全参数**：完整模型
- **推理模型** / **Thinking**：具有推理能力
- **Lite** / **Mini**：轻量版本
- **Turbo**：加速版本
- **Pro** / **Plus**：增强版本
